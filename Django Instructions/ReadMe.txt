1) Устанавливаем Python. Для этого запускаем файл python-3.4.3.msi или же скачиваем с офф. сайта: https://www.python.org/downloads/
2) После установки нужно прописать пути. 
 

 

 
Добавляем в "Значение переменной:" -> C:\Python34\;C:\Python34\Scripts;C:\Python34\Lib\site-packages\django\bin

 

3) Открываем cmd и переходим в папку с get-pip.py. 
Пример: "cd C:\Kormushka\kormushka\Django Instructions"
После устанавливаем pip командой "python get-pip.py"
4) Пишем в консоли "pip install virtualenv", ждем окончания установки. Далее нам нужно запустить проект на виртуальной машине, для этого переходим в папку в которой будет хранится проект и пишем "virtualenv venv". Виртуалка создана, теперь нужно её активировать, заходим в папку venv и пишем 
".\Script\activated" после этого перед папкой в которой мы находимся в скобках будет указана виртуалка (в данном случае venv). Для деактиации прописываем ".\Script\deactivated" 
5) Активируем виртуалку. Создаем проект командой: 
"python .\Script\django-admin.py starproject test_project"
В папке venv появилась новая папка test_project.
6) Теперь запускаем сервер, заходим в папку test_project и пишем в консоли: "python manage.py runserver" результат должен быть таким: 
 
Для проверки зайдем в браузере на 127.0.0.1:8000
 
7) Пишем в консоли: "python manage.py syncdb" на запрос пишем "yes", далее нас просят создать Логин и Пароль к базе.
